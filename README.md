# Glory Soccer
Glory Soccer is a 2D, pong-like soccer game built in Unreal Engine 4.

## Download


## Controls

              | Player 1      | Player 2
------------- | ------------- | -------------
**Move**          | W, A, S, D    | Up, Down, Left, Right
**Speed Boost**   | L-Shift       | R-Shift
